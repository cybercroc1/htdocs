Менеджер загрузок

Статусы загрузок:
-- "Загружается..." - загрузка еще не завершилась и происходит в фоновом режиме, отменить загрузку нельзя.
-- "Готово" - Загрузка успешно завершена.
-- "Ошибка" - Загрузка прервана из-за критической ошибки, которая скорее всего требует вмешательства разработчика.

Данные статистики загрузок:
-- Дата загрузки - дата начала загрузки
-- Файл, Имя - имя файла, из которого загружались данные
-- Файл, Байт - размер файла в байтах, из которого загружались данные
-- Загружено строк
-- Загружено телефонов
-- Одобрено строк - кол-во записей в рамках одной загрузки, которые разрешены для обработки оператором
-- Одобрено телефонов - кол-во телефонов в рамках одной загрузки, которые разрешены для обработки оператором
-- Отклонено пустых - кол-во записей, отклоненных при загрузке по причине отсутствия данных во всей строке или в одном из обязательных полей
-- Отклонено дубликатов - кол-во записей, отклоненных при загрузке по причине нарушения уникальности данных в зависимости от настроек исходных полей БД (см. соответствующий раздел) Напоминание: уникальность проверяется ТОЛЬКО в момент загрузки в соответствиe с текущими настройками исх. полей. Уникальность не ограничивается текущей загрузкой, а проверяется в рамках всего проекта и загружаемого файла.

Просмотр загруженных данных.
Для быстрого просмотра загруженных данных укажите в поле "Показывать строк" количество строк, которые нужно отобразить (по умолчанию 100) и нажмите на статус интересующей загрузки. Данные отобразятся в новом окне.

Выгрузка загруженных данных.
Можно выгрузить все или только одобренные записи. Если выбраны только одобренные записи, то выгрузятся только одобренные записи, а в поле "Телефон" будут присутствовать только одобренные номера.
-- Выгрузка в .xls(HTML)
Выгрузка файла ввиде HTML-таблицы с расширением .xls, такой файл открывается экселем по-умолчанию, но выгружается дольше, и занимает больше места, чем .csv

-- Выгрузка в .csv
Выгрузка в текстовый файл с расширением .csv (разделители: ";", ограничитель текста: ").

-- Выгрузка номеров в txt
Выгружаются только номера телефонов в один столбец (в одной строке одино номер), расширение файла .txt

Одобрение записей для обработки оператором
-- Одобрить роботом. Позволяет поменять статус записей и телефонов по списку выгруженному из робота.
-- Выберите одну или несколько загрузок, в которых нужно одобрить записи и нажниме "одобрить роботом", в открывшееся окно вставьте номера из робота или из другого источника.
СТРОГО: номера должны иметь вид 8хххххххххх (11 цифр) для региона "+7" и 810xxxxxxxxxxx...... (14-23 цифр) для международных номеров;
допускаются разделители: перенос строки, запятая, точка с запятой; допускаются пробелы (они игнорируются). Других смволов или форматов не допускается!

--Удаление загрузок
-- Выберите одну или несколько загрузок, которые хотите удалить. Из выбранных загрузок удалятся только не обработанные записи (с пустым статусом).
В случае, если в загрузке были обработанные данные, которые не удалились, то после удаления пересчитывается статистика загрузки. Иначе загрузка удаляется полностью.
После удаления загрузок автоматически пересчитываются квоты по исходным полям.
