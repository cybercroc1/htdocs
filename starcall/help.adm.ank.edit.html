Редактирование анкеты.

-- Страница - элемент анкеты 1-го уровня.
Автоматически добавляется в начале анкеты.
Страница содержит заголовок длинной до 4000 символов.
На странице размещаются группы и объекты, единовеменно доступные для чтения и ввода, переход на следующую страницу и сохранение введенных на странице данных осуществляется по кнопке "Далее".

-- Группа объектов - элемент анкеты 2-го уровня.
Автоматически добавляется в начале анкеты.
Группа объектов содержит заголовок длиной до 4000 символов (это загадочное число так часто появляется в инструкции потому, что в СУБД ORACLE - это максимальная длина типа данных varchar2:).
Группа объединяет объекты; на странице может быть размещено несколько групп объектов; в пределах группы можно включить ротацию объектов (вопросов) и еще что-нибудь... пока не придумал:)

-- Объект - элемент анкеты 3-го уровня.

Типы объектов:
Сообщение - информационное сообщение для оператора, только отображается на экране, в отчете не участвует. В нем можно использовать переносы строки и HTML-тэги для изменеия параметров шрифта. Имеет атрибут "сообщение" - это текст который выводится на экран. Максимальная длина сообщения 4000 символов.

Вопросы. Объект-вопрос имеет атрибуты:
"сообщение" - комментарий для оператора, относящийся к вопросу. Допускает переносы строки и HTML-теги.
"имя" - текст вопроса, видимый оператору и выводимый в отчете. Не допускает переносов строки и тегов.
"кодовое имя" - код вопроса, не видимый оператору, но выводимый в отчете, допускает любое значение без переносов строки и тегов.
признак обязательности - вопрос обязателен для ответа. Оператор не сможет продолжить опрос, если не ответин на обязательный вопрос.
"ротация" - только для вопросов с выбором из списка ответов. Отображение вариантов ответа в упорядоченном или случайном порядке.
Вопрос - Текст - одна строка для ввода произвольного текста. Максимальная длина ответа 4000 символов.
Вопрос - Большой текст - область для ввода произвольного текста, отличается от предыдущего тем, что визуально вмещает большее количество текста и допускает ввод символов перевода строки. Удобно, если вопрос подразумевает длинный ответ. Максимальная длина ответа 4000 символов.
Вопрос - Выбор - выбор одного варианта ответа из списка значений. Имеет вид раскрывающегося списка с вриантами ответа.
Вопрос - Радио - выбор одного варианта ответа из списка занчений. Имеет вид подписанных радио-кнопок (при нажатии одной кнопки отжимаются все остальные, на манер старых радиоприёмников :).
Вопрос - Множ. выбор - выбор нескольких вариантов ответа из списка значений. Имеет вид раскрывающегося списка, в котором с нажатой клавишей ctrl можно выбрать несколько вариантов.
Вопрос - Галочки - выбор нескольких вариантов ответа в виде подписанных квадратиков с галочками.
Вопрос - Число - строка для ввода числового значения. Допускает ввод только числовых значений.
Вопрос - Дата - позволяет выбрать дату.
Вопрос - Время - позволяет выбрать время.

Концы. Объект-конец выводит информационное сообщение для оператора и завершает опрос. Имеет атрибут "сообщение" длинной до 4000 символов. При достижении в опросе объекта "Конец" вывод на экран последующих обектов прекращается, а кнопка "далее" превращается в кнопку "завершить".
Последним объектом анкеты обязательно должен быть конец, если это не так, то он автоматически добавляется при сохранении.
Конец УСПЕШНЫЙ - завершает опрос со статусом "успешный" и участвует в квотах. Автоматически добавляется в конец анкеты при сохранении.
Конец НЕЦЕЛЕВОЙ - завершает опрос со статусом "не целевой" - не участвует в квотах.

Ошибки, отменяющие сохранение сохранение анкеты.
-- "ОШИБКА: Нельзя называть вопрос зарезервированным словом "Квота".


Предупреждения, позволяющие продолжить или отменить сохранение анкеты.
-- "ПРЕДУПРЕЖДЕНИЕ: Будет удалено один или несколько объектов анкеты."
-- "ПРЕДУПРЕЖДЕНИЕ: Изменено количество или порядок квотируемых вопросов. Если продолжить сохранение, то проект будет приостановлен, а квоты по вопросам перестроить и прописать значения заново частично или полностью."
